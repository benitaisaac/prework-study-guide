HTML

  • The head element contains information about the webpage
  • The body element shows content visible to the user
  • The HTML of a webpage can be compared to creating the foundation of a house
  • One way to describe elements within HTML is to refer to nested elements as "child" elements.

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: Adds starter code to current working branch
  • git commit -m "ENTER COMMIT MESSAGE": to commit changes
  • git pull origin main: to pull sources from Github
  • git push origin feature/starter-code: to push code to Github

Javascript

  • A variable is a container that allows us to store data in our code
  • Control flow order is the order in which a computer executes code in a script.
  • Functions are set of instructions that tell the computer to perform a certain task
  • In order to use a function in your code, you must both define and then call the function
